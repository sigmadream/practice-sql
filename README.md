# Practice SQL

## Install

```
// for macOS
$ brew install postgresql pgadmin4
$ brew services restart postgresql

//for windows
$ scoop install postgresql
$ pg_ctl start | stop | --help
$ del /s /f /q $HOME\apps\postgresql\current\data\
$ initdb -U postgres
$ psql -U postgres
psql (14.5)
도움말을 보려면 "help"를 입력하십시오.
postgres=#

```
